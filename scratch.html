TSP w/ Branch and Bound

Here is a youtube video that was SUPER helpful in teaching me about the branch and bound algorithm and how it can be used to solve the traveling salesman problem!

Steps to solve TSP with Branch and Bound:

  1. Matrix Reduction
    • Make sure each row and column has a zero value
    • Add together cost of adding zeros

  2. Exploring Chilfren
    • In the child matrix set the parent row and child column all to infinity and then reduce
    • Explore children of whichever child had the cheapest cost

  3. Repeat until all children have been explored
